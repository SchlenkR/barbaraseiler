2. Die Atemübung auf dieser Seite

Die Atemübung läuft vollständig in Ihrem Browser. Der optional zuschaltbare Klang (rosa Rauschen) wird client-seitig synthetisiert; es werden keine Audiodateien geladen. Es wird kein Mikrofon verwendet, keine Audiodaten werden aufgenommen oder übertragen.

Um Sie bei wiederkehrenden Besuchen nicht erneut nach der Atemminute zu fragen, wird nach vollständiger Übung ein client-seitiges Flag im localStorage Ihres Browsers gesetzt (Schlüssel: v36-atemraum:completed). Dieses Flag verbleibt ausschließlich auf Ihrem Gerät und wird zu keinem Zeitpunkt an einen Server übertragen. Sie können es jederzeit über die Einstellungen Ihres Browsers löschen.
